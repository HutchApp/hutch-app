Firefox

{{#if firefoxDownloadUrl}} Download Hutch for Firefox (test mode)

Installation steps

  1. Download hutch.xpi using the button above
  2. Firefox will prompt you to allow the installation — click "Continue to Installation"
  3. Click "Add" to install the extension

Login uses your real hutch-app.com account via OAuth. The version 0.0.1 saves the links in-memory and are lost after extension reloads. We're waiting the Firefox review to complete which may take some time.

{{else}}

The Firefox extension is not available for download yet. Please check back soon.

{{/if}}

Chrome

{{#if chromeDownloadUrl}} Download Hutch for Chrome

Installation steps

  1. Download hutch.zip using the button above
  2. Open chrome://extensions and enable "Developer mode"
  3. Drag and drop the downloaded zip onto the extensions page
{{else}}

The Chrome extension is not available for download yet. Please check back soon.
