Download Hutch for Firefox (test mode)

Installation steps

  1. Download hutch.xpi using the button above
  2. Firefox will prompt you to allow the installation — click "Continue to Installation"
  3. Click "Add" to install the extension

Login uses your real hutch-app.com account via OAuth, but saved links are stored in memory only and will be lost when the extension reloads.
